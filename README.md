# Istallazione software automatica 

Riepilogo dei software inclusi:
Sistema: htop (monitoraggio risorse), neofetch (info sistema), git & build-essential (per programmatori).

Internet: Firefox (browser), Thunderbird (email), FileZilla (trasferimento file), Transmission (torrent).

Grafica: GIMP (fotoritocco), Inkscape (vettoriale).

Multimedia: VLC (video player universale), Audacity (audio), OBS Studio (registrazione schermo e streaming).

Produttività: LibreOffice (suite completa per l'ufficio).

Istruzioni rapide:
Salva il testo in un file chiamato install.sh.

Dai i permessi: chmod +x install.sh.

Eseguilo con: sudo ./install.sh.
